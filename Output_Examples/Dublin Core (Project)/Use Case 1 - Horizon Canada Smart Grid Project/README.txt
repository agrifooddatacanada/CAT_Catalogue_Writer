SCENARIO
========

Horizon Canada, Canada's flagship €12M research and innovation funding program, funds a major international consortium project on "Sustainable Smart Grid Technologies." The 5-year project (2020-2025) spans 8 countries and generates massive research outputs across multiple formats and channels:

• 287 peer-reviewed publications (open access + subscription)
• 43 datasets (from field monitoring, simulations, surveys)
• 18 software tools (algorithms, optimization models, visualization platforms)
• 124 project documents (progress reports, policy briefs, training materials)

The project coordinator (led by Horizon Canada with contributors Niraj Patil, Keanu Smith, and Will Reeves) needs to:
1. Report annually to the Canadian Commission on all project outputs
2. Enable researchers worldwide to discover any project output or dataset
3. Track which outputs have been permanently archived vs. at-risk
4. Demonstrate project impact (publications, partnerships, policy adoption)
5. Manage project versions as outputs are updated, improved, or superseded

Instead of manually compiling spreadsheets, emailing progress reports, and fielding discovery requests, the project coordinator uses the Catalogue Record Writer tool:
- Opens the tool and selects "Dublin Core - Project Registry Profile"
- Answers a guided questionnaire (~20 minutes) with project information:
  * Project title: "Sustainable Smart Grid Technologies"
  * Project type: Multi-output registry (Software, Dataset, Text, InteractiveResource)
  * Creator/funder: Horizon Canada
  * Contributors: Niraj Patil, Keanu Smith, Will Reeves (project management)
  * Creation date: October 19, 2020 (project start)
  * Modified date: July 21, 2025 (latest registry update)
  * Project period: October 2020 to October 2025 (temporal coverage)
  * Geographic scope: Canada (with 8-country consortium)
  * Project identifier: https://horizon-canada.org/projects/101056789
  * Project extent: 287 publications, 43 datasets, 18 software tools, 124 documents (quantified output)
  * Output formats: EXE (executable), PY (Python), JPEG (visualizations), MP4 (videos), PDF (reports)
  * Accrual methods: Deposit (researchers submit outputs), Item Creation (project generates data)
  * Version tracking: v1.0, v1.2, v1.3 (registry evolves as new outputs added)
  * License: CC-BY 4.0 for project coordination + individual output licenses
  * Subject keywords: Smart Grid Technology, Renewable Energy, Urban Sustainability
  * Related initiatives: European Green Deal, iMaster project work plan

- Downloads standards-compliant JSON-LD record

The tool validates the project metadata, links all 287+43+18+124 individual outputs through the registry, structures version history, and prepares comprehensive reporting for funders, impact assessments, and global research discovery.
