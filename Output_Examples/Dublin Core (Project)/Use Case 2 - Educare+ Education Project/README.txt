SCENARIO
========

Educare+ is a strategic partnership initiative focused on digital literacy training for older adults. The three-year program (April 2019 – October 2022) spans 8 countries and trains 12,400 older adults in basic digital skills (email, social media, cybersecurity, online shopping, telehealth access).

Program outputs include:
• 12 comprehensive curriculum frameworks (by country/language)
• 67 modular training units (15-45 minutes each; self-paced or instructor-led)
• 15 evaluation and impact reports (documenting outcomes, challenges, lessons learned)
• Multiple formats: Videos (MP4), interactive tutorials (HTML), downloadable guides (PDF), infographics (JPEG)

The program is coordinated by Educare+ (Canada) with instructors from partner organizations in 8 countries (Niraj Patil, Instructor One, Instructor Two, Instructor Three leading).

Program Coordinator's Challenge:
1. Track outputs quarterly—which modules were created? Where? By whom?
2. Show impact at program conclusion: "How many older adults trained? In which countries? Which modules were most effective?"
3. Enable other care facilities to reuse training modules (licensing, discovery)
4. Demonstrate value to funders (Erasmus+ and national education ministries)
5. Support future funding for "Digital Aging" programs (evidence-based case)

Instead of collecting quarterly progress reports via email, managing spreadsheets across 8 countries, and manually compiling final evaluation, the program coordinator uses the Catalogue Record Writer tool:

- Opens the tool and selects "Dublin Core - Educational Training Program Profile"
- Answers a guided questionnaire (~18 minutes) with program information:
  * Program title: "Digital Literacy Training for Older Adults"
  * Program type: Educational resource collection (MovingImage, Sound, Text, InteractiveResource)
  * Creator/lead: Educare+
  * Contributors: 4 partner instructors (Niraj Patil, Instructor One, Instructor Two, Instructor Three)
  * Creation date: April 15, 2019 (program launch)
  * Modified date: November 16, 2020 (final update before program conclusion)
  * Program duration: April 2019 to October 2022 (temporal coverage)
  * Geographic scope: Guelph Ontario, Waterloo Ontario (Canada pilot sites, expanded to 8 countries)
  * Program identifier: https://education-canada.ca/projects/3649/4546732
  * Program extent: 12 curriculum frameworks, 67 training units, 15 evaluation reports
  * Output formats: JPEG (infographics), MP4 (video tutorials), PDF (curricula), HTML (interactive)
  * Accrual methods: Deposit (partner organizations contribute completed modules), Item Creation (Educare+ generates evaluation reports), License (CC-BY-SA for open reuse)
  * Version tracking: v1.0 (initial modules), v2.0 (refined after field testing)
  * License: CC-BY-SA 4.0 (allows partners and care facilities to adapt modules for local contexts)
  * Subject keywords: Digital Literacy, Older Adults, Active Aging, Social Inclusion
  * Related initiatives: Erasmus+ Digital Education action plan
  * Beneficiary data: 12,400 older adults trained across 8 countries

- Clicks "Export as JSON-LD"
- Downloads standards-compliant JSON-LD record

The tool structures the program metadata, links all 67 modules with curriculum frameworks, tracks version history, documents beneficiary reach, and prepares impact reports for funders and future program planning.
