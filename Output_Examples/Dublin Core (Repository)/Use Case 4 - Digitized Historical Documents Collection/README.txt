SCENARIO
========

The University of Guelph Library completes a major digitization project of historical primary sources. The project digitizes 1,247 documents from Canada's 1987 Archaeological Survey, including government correspondence, newspaper clippings, historical manifestos, and field reports. The collection spans the period when archaeological surveying practices were being standardized in Canada.

The library (administered by its cataloging team with contributors Niraj Patil, Johnson Dwayne, and Mike Ross) needs to:
1. Make these primary sources discoverable to historians, journalists, and researchers worldwide
2. Preserve the collection in multiple digital formats (TIFF preservation, JP2 web-accessible, OCR-generated text for searching)
3. Document the digitization process and archival provenance
4. Enable high school and university teachers to find materials for instruction
5. Ensure the collection is properly catalogued as a Canadian cultural heritage resource
6. Track version updates as materials are re-digitized or corrected

Instead of creating separate metadata entries in the library's catalog, the national archives system, and academic databases, the library uses the Catalogue Record Writer tool:
- Opens the tool and selects "Dublin Core - Historical Collection Profile"
- Answers a guided questionnaire (~15 minutes) with collection information:
  * Collection title: "1987 Archaeological Survey Digitized Document Collection"
  * Alternative title: "Digitized 1,247 documents from the 1987 Archaeological Survey"
  * Collection type: Collection (DCMI Type Vocabulary)
  * Creator/Institution: University of Guelph Library
  * Contributors: Niraj Patil, Johnson Dwayne, Mike Ross (digitization/cataloging staff)
  * Creation date: January 16, 2017 (when digitization project started)
  * Issued/publication date: December 22, 2017 (when collection went live)
  * Languages: English and French (bilingual Canadian documents)
  * Spatial coverage: Canada (geographic scope)
  * Subject: 1987 Archaeological Survey (primary subject)
  * Formats: TIFF (archival preservation), JP2 (web access), OCR-generated text (searchability)
  * Identifiers: UoG institutional ID + alternative research.ca ID
  * Version tracking: v2.1 (indicates ongoing improvement)
  * Provenance: "Original documents in UoG Library archives. Digitization 2016-2017."
  * Rights: "Public domain; see individual item metadata"
  * Part of: Canadian national archives infrastructure
- Clicks "Export as JSON-LD"
- Downloads standards-compliant JSON-LD record

The tool validates the format descriptions, structures the collection as an aggregation of 1,247 individual items, documents the digitization workflow, and prepares metadata for harvest by the Library of Congress, Library and Archives Canada, and international digital heritage networks.

The collection becomes a living, maintained digital asset with transparent version history—not a static artifact that decays over time.

Impact: Collection quality improved iteratively; researcher reliability enhanced through version tracking; archival transparency demonstrated; digital stewardship professionalized.
